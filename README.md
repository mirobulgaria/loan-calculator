# Кредитен калкулатор (статично уеб приложение)

Кредитен калкулатор с:
- Полета за **сума на кредита**, **годишна лихва (%)** и **срок (месеци)**
- Резултат за **месечна вноска**
- Пълен **погасителен план по месеци**
- По избор: **изтегляне на CSV** с плана

## Стартиране

### Вариант A (без инсталации): отваряне директно

Отворете `index.html` в браузър.

Ако двойният клик не го отваря, стартирайте:

```powershell
.\open.cmd
```

### Вариант B (препоръчително): локален сървър

От тази папка изпълнете:

```powershell
python -m http.server 5173
```

После отворете `http://localhost:5173`.

### Готови стартери

- `start.cmd`: стартира сървъра и отваря браузъра
- `start.ps1`: същото, но за PowerShell

Ако при двойно кликване “нищо не става”, стартирайте от терминал, за да видите грешките:

```powershell
cmd /k .\start.cmd
```

PowerShell може да блокира `.ps1` по подразбиране; това го стартира еднократно без промяна на системна политика:

```powershell
powershell -ExecutionPolicy Bypass -File .\start.ps1
```

И двата стартера записват лог файл: `launcher.log`.
